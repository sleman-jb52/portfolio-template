# Portfolio Template

Thank you for purchasing template.

## Getting Started

Open `index.html` in your browser or use a local server.

## Editing Content

Most Content is loaded from `data.json`.
Simply edit the values to customize the template.

## Hero Profile Image

To change the hero profile image

Recommended:

1. Use a square image (1:1 ratio)
2. Recommended size: 600x600px or higher
3. Face centered vertically

## Screenshots Section

Images are loaded Dynamically using Swiper.
Add or remove images in `data.json`.

## Notes

1. If your hero image looks cropped, adjust `object-position`
   inside `.hero-avatar img` in CSS
2. If your image appears slightly cropped, this is normal for circular profile images.
   For best results, use a centered square photo.
